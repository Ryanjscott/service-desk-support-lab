# Printer Troubleshooting Guide

## Purpose
This guide provides a basic Level 1 process for troubleshooting common printer issues on Windows devices.

## Common Symptoms
- Print jobs stuck in queue
- Printer appears offline
- User cannot see shared printer
- Printer does not respond
- Test page fails to print

## Initial Checks
1. Confirm the printer has power.
2. Confirm the printer has paper and toner.
3. Check for visible error messages.
4. Confirm whether the issue affects one user or multiple users.
5. Confirm the user is connected to the correct network.
6. Check the Windows print queue.

## Common Fixes
- Cancel stuck print jobs
- Restart the Windows Print Spooler service
- Remove and re-add the printer
- Restart the printer
- Restart the user device
- Send a test print after changes

## Escalation Criteria
Escalate if:
- Multiple users cannot print
- Printer hardware shows a fault
- Printer is unreachable on the network
- Driver reinstall requires admin permissions
- Print server issue is suspected

## Notes
Confirm whether the issue is user-specific, device-specific or printer-wide before escalating.
