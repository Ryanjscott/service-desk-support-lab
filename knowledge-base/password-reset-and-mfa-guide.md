# Password Reset and MFA Troubleshooting Guide

## Purpose
This guide provides a basic Level 1 process for troubleshooting Microsoft 365 sign-in issues after a password change or MFA prompt issue.

## Common Symptoms
- User cannot sign in to Outlook, Teams or OneDrive
- User recently changed their password
- MFA prompt is not appearing
- Sign-in works in browser but not desktop apps
- User receives repeated password prompts

## Initial Checks
1. Confirm the user has internet access.
2. Confirm whether the issue affects one Microsoft 365 app or all services.
3. Ask whether the user recently changed their password.
4. Ask whether MFA is appearing and being approved.
5. Test sign-in using a private browser window.
6. Confirm whether the issue occurs on another device or browser.

## Common Fixes
- Clear cached credentials
- Restart the affected application
- Re-authenticate Microsoft 365 apps
- Confirm password is correct
- Retry MFA approval within the required time window
- Restart the device if authentication issues continue

## Escalation Criteria
Escalate if:
- The user account appears locked or disabled
- MFA device is lost or unavailable
- The user cannot access any Microsoft 365 service after basic checks
- Multiple users report similar sign-in issues
- Admin permissions are required

## Notes
Always confirm the user’s access has been restored before closing the ticket.
