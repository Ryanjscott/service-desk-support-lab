# Service Desk Support Lab

## Overview
This project is a simulated Level 1 Service Desk / MSP support lab. It demonstrates ticket triage, troubleshooting, customer communication, escalation notes, resolution documentation and knowledge base creation.

The project reflects common entry-level IT support tasks across Windows, Microsoft 365, networking, printers, remote access, security awareness and onboarding.

## Tools Used
- Jira Service Management
- GitHub
- Windows 10/11 troubleshooting tools
- Command Prompt and PowerShell basics
- Microsoft 365 support documentation
- Technical documentation and knowledge base writing

## Skills Demonstrated
- Ticket logging and prioritisation
- Customer communication
- Troubleshooting methodology
- Documentation and closure notes
- Escalation decision-making
- Basic Windows, Microsoft 365 and networking support
- Knowledge base creation
