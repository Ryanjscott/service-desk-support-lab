# Ticket 009: New Starter Device Setup

## Summary
New employee required a laptop, user account access, Microsoft 365 setup and standard applications before their start date.

## Category
Onboarding / Device Setup

## Priority
Medium

## Impact
New starter required a prepared device and access before commencing work.

## Initial User Update
Hi, I’ve received the new starter setup request. I’ll prepare the device, confirm required access, install standard applications and document the setup before handover.

## Internal Triage Note
Onboarding request. Tasks included device preparation, access confirmation, Microsoft 365 setup, standard software installation, asset record update and handover checklist.

## Setup Steps
1. Confirmed new starter name, department and start date.
2. Confirmed required applications and access requirements.
3. Prepared Windows laptop and checked device condition.
4. Confirmed Windows updates were installed.
5. Installed standard applications.
6. Confirmed Microsoft 365 applications were available.
7. Updated asset register with device details.
8. Prepared basic handover notes for the user.

## Resolution
New starter device prepared, standard applications installed, Microsoft 365 access confirmed and asset register updated. Device ready for user handover.

## Escalation Required
No.

## Closure Note
Onboarding setup completed successfully.
