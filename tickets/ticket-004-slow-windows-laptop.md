# Ticket 004: Slow Windows Laptop

## Summary
User reported their Windows laptop was running slowly, freezing intermittently and taking longer than usual to open applications.

## Category
Windows / Performance

## Priority
Medium

## Impact
User productivity was affected, although the device remained usable.

## Initial User Update
Hi, I’ve received your request about the slow laptop. I’ll check system performance, startup applications, available storage and recent updates to identify the likely cause.

## Internal Triage Note
Issue appeared limited to one Windows device. Initial checks focused on resource usage, storage capacity, startup applications, Windows updates and possible background processes.

## Troubleshooting Steps
1. Confirmed the issue was affecting one device only.
2. Checked Task Manager for high CPU, memory and disk usage.
3. Reviewed startup applications and disabled unnecessary startup items.
4. Checked available disk space.
5. Confirmed Windows updates were pending.
6. Restarted the device after updates completed.
7. Verified performance improved after reducing startup load and completing updates.

## Resolution
Completed pending Windows updates, reduced unnecessary startup applications and restarted the device. User confirmed the laptop was responding normally after the changes.

## Escalation Required
No.

## Closure Note
Issue resolved. Likely cause was pending updates and excessive startup/background processes.
