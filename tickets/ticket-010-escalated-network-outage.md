# Ticket 010: Escalated Network Outage

## Summary
Multiple users were unable to access network services, including shared drives and internal applications.

## Category
Network / Escalation

## Priority
Critical

## Impact
Multiple users were affected. Internal applications and shared drives were unavailable, creating wider business impact.

## Initial User Update
Hi, we’ve received reports of a wider network access issue. I’m completing initial checks and escalating this to the appropriate support team due to the number of affected users. I’ll provide updates as more information becomes available.

## Internal Triage Note
Issue affected multiple users and services. Initial checks focused on confirming scope, affected services, network connectivity, recent changes and whether escalation to senior/network support was required.

## Troubleshooting Steps
1. Confirmed the issue affected multiple users.
2. Confirmed shared drives and internal applications were unavailable.
3. Confirmed affected users were connected to the network.
4. Checked whether internet access was still available.
5. Confirmed the issue was not isolated to one device.
6. Gathered affected user examples and impacted services.
7. Escalated to network/senior support due to wider business impact.
8. Continued user updates while escalation was in progress.

## Resolution
Escalated to network/senior support with affected users, impacted services and initial troubleshooting notes. Users were kept informed while investigation continued.

## Escalation Required
Yes.

## Closure Note
Escalated due to wider network impact. L1 triage confirmed issue was not isolated to a single user or device.

## Skills Demonstrated
- Incident triage
- Network issue scoping
- Escalation decision-making
- User communication during wider outage
- Priority and impact assessment
